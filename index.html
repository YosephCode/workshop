Ycode workshop

O Ycode workshop foi criado no dia 01/07/2015, com intuito de atender a crescente demanda de designers, publicitários, desenvolvedores e pessoas interessadas em começar seu próprio negócio na Web.

Conteúdo

O conteúdo do workshop está separado em módulos para uma melhor compreenção de cada participante sobre as tecnologias ensinadas. Seguem os módulos do workshop:

  • Módulo 1: HTML, CSS, Frameworks CSS, Design Responsivo e Mobile First
  • Módulo 2: (PROJETO) HTML, CSS, Framework CSS, Design Responsivo e Mobile First
  • Módulo 3: HTML, CSS, Javascript, AngularJs e Jquery.
  • Módulo 4: Javascript, Grunt, Bower, Github.
